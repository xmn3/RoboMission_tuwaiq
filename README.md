## 📌 Description
RoboMission is a robotics project built using the mBot platform.
The robot is designed to complete specific missions such as navigating paths,
avoiding obstacles, and performing automated tasks.

This project was developed as part of a robotics learning journey in Tuwaiq Academy.

---

## 🚀 Features
- Autonomous movement
- Obstacle detection using ultrasonic sensor
- Line tracking capability
- Mission-based programming
- Real-time response to environment

---

## 🛠️ Components Used
- mBot Robot Kit
- Ultrasonic Sensor
- Line Follower Sensor
- Motors & Wheels
- Arduino-based Controller

---

## ⚙️ How It Works
The robot uses sensors to detect its surroundings.
Based on the input, it makes decisions like stopping, turning, or continuing forward.
It can follow lines and avoid obstacles to complete its mission.

---

## ▶️ How to Run
1. Connect the robot to your computer
2. Upload the code using mBlock / Arduino IDE
3. Turn on the robot
4. Place it on the track
5. Start the mission
